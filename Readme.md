# **SignUp**

## **Overview/Project Summary**
This app creates sign up lists for various events. 


## **Technical Summary**
* Backend: C#
* Framework: ASP.NET Core MVC
* ORM: EF Core
* Server: IIS
* DB: SQL
* Front End: HTML/CSS, Bootstrap/ JQuery

## **Getting Started**
* Clone the project
* Build the application
* Enjoy Creating signup lists!
